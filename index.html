WebAssembly-powered graphics with native-like performance in the browser.

Full 2D canvas support with shapes, paths, text rendering, and custom colors.

Mouse and keyboard events with trait-based handling for clean, reusable code.

Built-in requestAnimationFrame support for smooth 60fps animations.

Complete type definitions for seamless TypeScript integration.

🛠️ Easy Integration

Drop-in JavaScript files for quick integration into any web project.

🚀 Quick Start

Get started with just a few files from the dist/ directory:

// app.js import { loadWasm } from './wasm-utils.js'; import { getCanvasImports } from './canvas-wasm.js'; document.addEventListener('DOMContentLoaded', async () => { const wasmExports = await loadWasm('./your-game.wasm', getCanvasImports()); wasmExports.main_function(); });
// Rust game logic use web_canvas::canvas; impl canvas::EventHandler for MyGame { fn on_animation_frame(&mut self, canvas: &canvas::Canvas, elapsed: f32) { canvas.set_fill_color(canvas::BLACK, 1.0); canvas.fill_rect(0.0, 0.0, canvas.width(), canvas.height()); // Draw your game! canvas.draw_circle(x, y, radius, canvas::WHITE); } }

📁 Required Files for Your Project

Copy these files from dist/ to integrate Web Canvas into your project:

  • canvas-wasm.js - Core canvas functionality and event handling
  • wasm-utils.js - WebAssembly utilities and browser integration
  • types/ - TypeScript definitions (optional)
  • your-compiled.wasm - Your Rust code compiled to WebAssembly
